## Задача 1. 
У вас есть социальная сеть, где пользователи (id, имя) могут ставить друг другу лайки. Создайте необходимые таблицы для хранения данной информации. 
Создайте запрос, который выведет информацию:
* id пользователя;
* имя;
* лайков получено;
* лайков поставлено;
* взаимные лайки.
